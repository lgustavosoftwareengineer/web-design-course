Section 2 - Fundamentos do HTML - Tabelas

Podemos criar tabelas utilizando a tag: '<'table'>',

suas linhas com a tag: '<'tr'>'

e os elementos das linhas com a tag: '<'td'>'

mas geralmente se é utilizado a tag '<'th'>' para declarar o líder das colunas ou a linha líder.

Ex.:

cédula 1.1 cédula 1.2
cédula 2.1 cédula 2.2
cédula 3.1 cédula 3.2
