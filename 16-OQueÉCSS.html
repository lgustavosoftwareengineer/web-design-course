Section 3 - Folhas de Estilos em Cascata

O que é CSS?

Conjunto de regras de formatação para estilo das páginas web

O nome CSS significa Cascading Style Sheet

Propósito chave

Separar conteúdo da aparência!

Ordem de prioridade para as mais diferentes formas de estilização (menor prioridade para maior prioridade):

  1. Folha de estilo padrão do navegador;
  2. Folha de estilo do usuário;
  3. Folha de estilo do desenvolvedor:
    1. Estilo externo (importado);
    2. Estilo incorporado (definido no head ou no body);
    3. Estilo inline (dentro de um elemento);
  4. Marcação !important;

Declarando códigos CSS

    Há três formas:

  1. Inline

    Feito diretamente na tag

    Ex.: '<'div style="color:red;">' Meu texto <'/div'>

  2. Tag '<'style'>'

    A estilização é feita através da tag <'style'> declarada dentro da tag <'html'>

    Ex.:

    <'head'>

    <'style>' p { color:red; } <'/style'>

    <'/head'>
    <'body'>

    <'p'> Meu texto <'/p'>

    <'/body'>

  3. Importação

    Estilo que é importado de um arquivo ".css"

    Ex.:

    <'head'>

    <'link' rel="stylesheet" type="text/css" href="style.css">

    <'/head'>

Sintaxe

A sintaxe é simples:

  • Há um seletor que pode ser uma classe, um id ou uma tag;
  • Há uma declaração que é a característica que quer ser alterada e há um valor que é como a característica será mudada

Ex.:

seletor{

declaração:valor;

}

Diferença entre Classes e Ids

  1. Ids
    • identificadores únicos;
    • declarados com #
    • Urls
      • Ex.: www.site.com/#id
  2. Classes
    • Apresenta um estilo a um grupo de elementos;
    • declarado com .;
